# C214 L1 - Projeto

## Como rodar?

### Pré-requisitos:

- Node v16.17 ou superior;
- Npm v8.15 ou superior;

### Passo a passo:

- Ir para o diretorio do repositório:
  ```
  cd c214l1-projeto/
  ```

- Instalar dependencias do Node:
  ```
  npm i
  ```

- Startar o servidor web:
  ```
  npm run dev
  ```
